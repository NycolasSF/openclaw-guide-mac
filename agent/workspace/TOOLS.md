# TOOLS.md — Notas locais

Skills definem _como_ as ferramentas funcionam. Este arquivo é para suas _especificidades_ — o que é único do seu setup.

## O que vai aqui

Coisas como:

- Nomes e localizações de câmeras
- SSH hosts e aliases
- Vozes preferidas para TTS
- Nomes de speakers/cômodos
- Apelidos de devices
- Qualquer coisa específica do ambiente

## Exemplos

```markdown
### Cameras

- living-room → Sala principal, 180° wide angle
- front-door → Entrada, motion-triggered

### SSH

- home-server → 192.168.1.100, user: admin
- vps-prod → mydomain.com, user: nyc

### TTS

- Voz preferida: "Nova" (calorosa, levemente britânica)
- Speaker default: HomePod da Cozinha

### Devices

- iPhone: Nyc-15
- Mac: Nyc-MacMini
```

## Por que separar?

Skills são compartilhadas. Seu setup é seu. Manter separado significa que você pode atualizar skills sem perder suas notas, e compartilhar skills sem vazar sua infraestrutura.
